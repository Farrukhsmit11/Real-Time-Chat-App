A modern real-time chat application built with the MERN Stack that enables users to communicate instantly with secure authentication and real-time messaging.

🚀 Features

- 🔐 JWT Authentication
- 💬 Real-time one-to-one messaging using Socket.io
- 👥 User list and private conversations
- 📩 Message history stored in MongoDB
- 🎨 Responsive and modern chat UI
- 🔒 Protected API routes
